# Day 2 – Networking Basics

## Overview  
This repository contains my Day 2 learning on Networking Fundamentals as part of my cybersecurity journey.

---

## Topics Covered  
- What is a Network  
- IP Address basics  
- DNS (Domain Name System)  
- Ports and protocols  
- Basic network commands  

---

## Commands Practiced  
```bash
ping google.com
nslookup google.com
tracert google.com   # Windows
traceroute google.com   # Linux
```

---

## Key Learning  
- Networks connect devices for communication  
- IP address uniquely identifies each device  
- DNS converts domain names into IP addresses  
- Ports act as communication endpoints  
- Commands help in basic network troubleshooting

This session helped me understand the basics of networking, which is essential for cybersecurity and SOC roles.
